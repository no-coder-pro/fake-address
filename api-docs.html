API Endpoints

Integrate our fake address generator directly into your application.

1. Get Country List

GET /api/countries

Returns a mapping of country codes to names and the total count of addresses available for each.

2. Get Random Address

GET /api/random

Returns a completely random address from any supported country.

3. Get Specific Country Address

GET /api/address?code={country_code}

Returns a random address for the specified country code (e.g., US, BD, XK).

4. Get Address by Name

GET /api/address?name={country_name}

Returns a random address for the specified country name. Supports fuzzy matching.
